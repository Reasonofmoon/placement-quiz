# Placement Quiz — Find Your Level

AI Engineering from Scratch 커리큘럼의 `/find-your-level` 배치 퀴즈를 브라우저에서 풀 수 있는 정적 웹앱입니다.

## Features

- 10문항 (5영역 × 2문항) 자가 진단
- 시작 Phase 추천 (Phase 1 ~ 14)
- 20 Phase 개인화 학습 경로 (Skip / Review / Do)
- localStorage 결과 저장
- Markdown 결과 복사

## Run locally

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080` in your browser.
